An Introduction to Palaeography Terminology

Like all specialist fields, palaeography has specific terms you’ll need to learn. For example, a script is the model of writing used by a scribe, like a modern font, whereas the hand is the specific handwriting of the scribe themselves. There are often many hands in a single manuscript writing in the same script. Similarly, one hand can write multiple scripts in the same manuscripts.

Since scholars of different scripts focus on different detials, there are often overlaping terms describing the same components. However, there are some universal terms which are important to know.

Aspect: The aspect of a script is its overall appearance. For example, is it rounded or angular? Tall and narrow or wide and squat?

Character: A single occurrence of a written item. This can be a letter, punctuation, or numbers.

Graph: A single occurrence of a letter of an alphabet.

Allograph: A recognised variant form of the same graph. For example ‘a’ and ‘a’.

Digraph: Two letters which refer to the same sound. For example, in Old English, þ and ð are digraphs. Similarly, ‘i’ and ‘y’ can be digraphs.

Ligature: Two letters joined by sharing the same stroke.

Component: A recognisable element of a character. For example, the bowl in ‘b’ and ‘p’.
