Patch Notes

Version 2.0. Changes include:

  1. HTML structure has been simplified and converted, where possible, to semantic HTML.
  2. Style guide created for new page creation and CSS updated and simplified in line with the new style guide.
  3. Converted all images containing text to HTML so as to be compatible with screenreaders and copyable by users.
  4. Minor design changes such as making the heading smaller, increasing font size and line height, and adjusting element positioning to ensure better UX.
  5. Updated grammar tables so they are properly responsive on mobile.
  6. Added hideable tooltip template to allow the addition of interesting but non-critical information without increasing visual complexity. See example here.
  7. Added abbrevation tag to complex words so they can be hovered for explanations.
  8. Added audio to the phonetics table to demonstrate correct pronounciation.
  9. Added the Weak Verbs section.
  10. Added grammar test and expanded the Demonstratives section of Strong Masculine Nouns.
  11. Added additional questions to the grammar tests of Cases to provide greater variety.
  12. Updated About Page to reflect completion of Masters and transition onto the development of the project as a stand-alone learning resource.
