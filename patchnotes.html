Patch Notes

Version 2.0. Changes include:

  1. HTML structure has been simplified and converted, where possible, to semantic HTML.
  2. Style guide created for new page creation and CSS updated and simplified in line with the new style guide.
  3. Converted all images containing text to HTML so as to be compatible with screenreaders and copyable by users.
  4. Minor design changes such as making the heading smaller, increasing font size and line height, and adjusting element positioning to ensure better UX.
  5. Expanded the Demonstratives section of Strong Masculine Nouns and added a grammar test.
  6. Increased the number of questions in the grammar test of each Cases module.
  7. Updated grammar tables so they are properly responsive on mobile.
  8. Added hideable tooltip template to allow the addition of information without increasing visual complexity. See example here.
  9. Added abbrevation tags to complex words so they can be hovered for explanations.
  10. Added audio to the phonetics table to demonstrate correct pronounciation.
  11. Added the Weak Verbs and Strong Neuter and Feminine Nouns modules.
  12. Updated About Page to reflect completion of Masters and transition onto the development of the project as a stand-alone learning resource.
  13. Created the patch notes page to track changes.
  14. Created the license page for licensing and copyright info.
